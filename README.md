# Papyrus Token Bridge web dApp

This repository is a copy of [POA Bridge UI v2.0.0](https://github.com/poanetwork/bridge-ui/tree/2.0.0). It is created as simple example of using Papyrus Token Bridge.

It will be very useful to read [POA Bridge UI documentation](https://github.com/poanetwork/bridge-ui/blob/2.0.0/README.md).

# How to use

To use web dApp it is necessary to configure file [papyrus/papyrus-bridge-web.env](papyrus/papyrus-bridge-web.env) and run docker container. To do that you need have [Docker](https://docs.docker.com/install/) and [Docker Compose](https://docs.docker.com/compose/install/) installed.

To run web dApp just run script [papyrus/start-server.sh](papyrus/start-server.sh):

```sh
papyrus/start-server.sh
```
